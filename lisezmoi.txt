Paquet de distribution de BATBOX.

Liste des sous-répertoires :

	* src     	: Sources de batbox (en asm pour Flat Assembler)

	* man		: Sources de la documentation (pour tea)

	* bin     	: Exécutables

	* doc	  	: Documentation

	* exemples	: Exemples d'utilisation

	* spritebox 	: Dossier de sources de spritebox
